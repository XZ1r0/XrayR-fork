# XrayR-fork

本仓库基于原作者 [XrayR-project/XrayR](https://github.com/XrayR-project/XrayR) 的项目进行 fork 和备份维护。

由于原仓库已经删库，本人重度使用 XrayR，为了后续自用和方便恢复部署，因此 fork 并保留此仓库。

如有侵权或不适合继续公开保存，请提交 issue 说明，我会及时处理或删除。

## 安装

```bash
wget -N https://raw.githubusercontent.com/XZ1r0/XrayR-fork/main/install.sh && bash install.sh install
```

## 卸载

```bash
wget -N https://raw.githubusercontent.com/XZ1r0/XrayR-fork/main/install.sh && bash install.sh uninstall
```
